# NEXUS Electron — Release Checklist

Follow this checklist before every public release (DMG or App Store update).

---

## 1 · Pre-build Setup

- [ ] `APPLE_ID`, `APPLE_APP_SPECIFIC_PASSWORD`, `APPLE_TEAM_ID` set in `.env`
- [ ] Code signing certificate (Developer ID Application) installed in Keychain Access
- [ ] `electron/build/icon.icns` generated from a 1024×1024 `icon.png`
- [ ] `electron/build/iconTemplate.png` + `iconTemplate@2x.png` present (22×22, monochrome, for menu bar tray)
- [ ] Version number bumped in `electron/package.json` → `"version": "X.Y.Z"`

---

## 2 · Build

```bash
# From nexus/ root:
npm run electron:build:dmg
```

Artifacts land in `electron/dist-app/`.

---

## 3 · Functional Testing (both Intel x64 and Apple Silicon arm64)

### App Launch
- [ ] App launches without a white flash (window shows only after `ready-to-show`)
- [ ] Window size/position from the last session is restored correctly
- [ ] Tray icon appears in the Mac menu bar immediately

### Authentication
- [ ] Google OAuth: clicking "Sign in with Google" opens the browser → user approves → app receives `nexus://auth/callback` deep link → user is logged in
- [ ] Email/password login works normally
- [ ] Session persists after quitting and reopening the app

### Widgets
- [ ] All widgets load and render correctly
- [ ] Spotify widget plays/pauses and displays the current track
- [ ] Shared Drawing Canvas real-time sync works (WebSocket still connects)
- [ ] Shared Photo widget camera access works (permission granted via entitlements)
- [ ] Calendar widget shows events (Google Calendar OAuth completed)
- [ ] F1 widget live timing displays during a race weekend

### Tray Panel
- [ ] Clicking the tray icon opens the 320×440 popover panel
- [ ] Clicking outside the panel dismisses it
- [ ] "Open NEXUS" button focuses the main window
- [ ] Spotify section shows the current track (if Spotify connected)
- [ ] Pomodoro section shows the correct time remaining and session type
- [ ] Next calendar event is shown with a countdown
- [ ] Habit checkboxes toggle correctly

### Native Integrations
- [ ] Global shortcut `Cmd+Shift+N` toggles the main window from another app
- [ ] Dock badge updates when there are unread notifications (test manually via `window.electronAPI.setDockBadge(3)`)
- [ ] Dock right-click shows "New Page" and "Start Focus Session" items
- [ ] Native macOS notification appears for a Pomodoro session end
- [ ] Clicking a notification focuses the correct widget
- [ ] Menu bar → NEXUS → Preferences… opens Settings modal
- [ ] Menu bar → File → New Page creates a new dashboard page

### Auto-Update
- [ ] Publish a test release to GitHub Releases with a higher version number
- [ ] App detects the update within 5 seconds of launch
- [ ] In-app update banner appears (not a modal — just a strip at the top)
- [ ] Clicking "Restart to update" quits and installs the new version

### Connectivity
- [ ] App works on both Intel (x64) and Apple Silicon (arm64) builds
- [ ] All API calls go to the correct backend (`nexus-api.lj-buchmiller.com`)
- [ ] No `mixed content` or SSL errors in the console

---

## 4 · Code Signing & Notarization

```bash
# Verify code signing
codesign --verify --deep --strict electron/dist-app/mac/NEXUS.app
echo $?   # must be 0

# Verify notarization (stapled ticket)
spctl --assess --type exec electron/dist-app/mac/NEXUS.app
echo $?   # must be 0
```

- [ ] `codesign --verify` exits 0
- [ ] `spctl --assess` exits 0 (Gatekeeper approves)
- [ ] Drag the DMG to another Mac (or a fresh user account) — app opens without "damaged" or "unidentified developer" warnings

---

## 5 · DMG Inspection

- [ ] DMG mounts correctly with the expected background and icon placement
- [ ] `NEXUS.app` inside the DMG opens cleanly (drag to Applications)
- [ ] After installation, the app appears in Spotlight and Launchpad

---

## 6 · GitHub Release

```bash
git tag vX.Y.Z && git push origin vX.Y.Z
```

- [ ] Draft release created on GitHub with the correct tag
- [ ] DMG file uploaded as a release asset (`NEXUS-X.Y.Z-arm64.dmg`, `NEXUS-X.Y.Z-x64.dmg`)
- [ ] `latest-mac.yml` update manifest uploaded alongside the DMG files (generated by electron-builder)
- [ ] Release published (not just a draft)

---

## 7 · Post-release

- [ ] Auto-update confirmed working from the previous version
- [ ] `nexus.lj-buchmiller.com` download link updated to point to the new DMG
- [ ] Version number bumped to next pre-release version in `electron/package.json`
